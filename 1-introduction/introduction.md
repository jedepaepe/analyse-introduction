# Introduction

En développement d'application, l'analyse consiste à analyser une application. Les buts sont variables en fonction de l'état de l'application.
Si l'application n'existe pas encore, qu'elle est à l'état de projet, le but de l'analyse sera de faciliter le développement de l'application.
Si l'application existe déjà et est en production, le but de l'analyse sera de faciliter la correction de bugs ou la modification de l'application.

Il est donc essentiel de bien comprendre le cycle de vie d'une application pour saisir le role de l'analyse tout au long de ce cycle.
